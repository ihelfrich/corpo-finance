MODULE // 01
MEGACORP GOVERNANCE PROTOCOLS
Corporate Finance & Governance
  • Goal of the firm: maximize shareholder value (NPV > 0)
  • Investment decisions vs. financing decisions
  • Agency problem: managers vs. shareholders
  • Free Cash Flow (FCF) = NOPAT − Net investment in OC
  • EVA = NOPAT − (WACC × Invested Capital); MVA = PV(EVA)
  • ESG as a long-run value driver
FCF = EBIT(1−T) − ΔNOWC − ΔNet PPE
WEEK 2
MODULE // 02
NETRUNNER DATA EXTRACTION
Financial Statement Analysis
  • DuPont decomposition: ROE = Profit Margin × Asset Turnover × Equity Multiplier
  • Liquidity, leverage, efficiency, profitability ratios
  • Additional Funds Needed (AFN) = spontaneous liabilities increase
  • Pro forma forecasting from % of sales
  • Book value vs. market value distinction
AFN = (A*/S₀)ΔS − (L*/S₀)ΔS − MS₁(1−d)
WEEK 3
MODULE // 03
TEMPORAL CREDIT CYCLES
Time Value of Money
  • FV = PV × (1+r)^t — compounding over t periods
  • PV = FV / (1+r)^t — discounting at rate r
  • Annuity PV = PMT × [1 − (1+r)^−n] / r
  • Perpetuity PV = C / r (no end date)
  • EAR = (1 + APR/m)^m − 1 ≠ APR when m > 1
  • Excel: PV(), FV(), PMT(), RATE(), NPER()
EAR = (1 + APR/m)^m − 1
WEEK 4
MODULE // 04
RISK CALIBRATION MATRIX
Risk, Return & CAPM
  • Diversification eliminates specific risk; market risk remains
  • Beta (β) = Cov(R_i, R_M) / Var(R_M) — slope of regression on S&P 500
  • CAPM: r_i = r_f + β(r_M − r_f)
  • Security Market Line (SML): expected return as linear function of β
  • β = 1 → market-like; β > 1 → aggressive; β < 1 → defensive
  • Only systematic (market) risk earns a risk premium
r_i = r_f + β × (r_M − r_f)
WEEK 5
MODULE // 05
ASSET PRICING ENGINE
Bond & Stock Valuation
  • Bond price = PV of coupons + PV of par (discount at YTM)
  • Price ↑ when YTM ↓; inverse relationship always holds
  • YTC: solve for r using call price at call date instead of par
  • Gordon (DDM): P₀ = D₁ / (r − g) when g < r
  • Supernormal growth: discount each phase separately, then add
  • P/E and EV/EBITDA as relative valuation cross-checks
P₀ = D₁ / (r − g) [Gordon Growth Model]
WEEK 6
MODULE // 06
HYBRID WARE CONTRACTS
Hybrid Securities
  • Preferred stock: fixed dividend, senior to common, no voting rights
  • Cost of preferred: r_ps = D_ps / P₀ (perpetuity, no growth)
  • Warrants: long-dated call options issued by the firm itself
  • Convertible bonds: debt with an embedded call on the stock
  • Hybrid = lower rate debt + option premium to issuer
  • Dilution effect when warrants/converts are exercised
r_ps = D_ps / P₀(1−F) [preferred cost of capital]
WEEK 7
MODULE // 07
PROJECT DELTA CLEARANCE
Capital Budgeting: NPV, IRR, MIRR
  • NPV = Σ[FCF_t / (1+r)^t] — the correct universal rule
  • IRR: set NPV=0 and solve for r; accept if IRR > WACC
  • MIRR: reinvest CFs at WACC, finance neg. CFs at cost of capital
  • Payback, Discounted Payback: secondary screens (ignore distant CFs)
  • Profitability Index = NPV / Cost (for capital rationing)
  • EAA: annualize NPV for unequal lives comparison
NPV = Σ FCF_t/(1+r)^t − C₀ > 0 → ACCEPT
WEEK 8–9
MODULE // 08
WEIGHTED NEURAL NETWORK COST
WACC
  • WACC = w_d × r_d(1−T) + w_ps × r_ps + w_e × r_e
  • Weights MUST be market-value weights, not book values
  • Cost of equity: r_e via CAPM = r_f + β(r_M − r_f)
  • Cost of debt: r_d = YTM on existing bonds (after-tax = r_d × (1−T))
  • WACC is the minimum acceptable return on average-risk projects
  • Use firm WACC only when project risk ≈ firm's existing risk
WACC = w_d r_d(1−T) + w_ps r_ps + w_e r_e
WEEK 10
MODULE // 09
LEVERAGE OPTIMIZATION PROTOCOL
Capital Structure: M&M & Hamada
  • M&M I (no tax): V_L = V_U — capital structure is irrelevant
  • M&M II (no tax): r_e rises with leverage to offset cheaper debt
  • M&M with taxes: V_L = V_U + T×D (interest tax shield adds value)
  • Trade-off theory: optimal D/E balances tax shield vs. distress costs
  • Pecking order: prefer retained earnings > debt > equity
  • Hamada: β_L = β_U [1 + (1−T)(D/E)] — re-lever / unlever beta
β_L = β_U × [1 + (1−T)(D/E)] [Hamada equation]
WEEK 11
MODULE // 10
EDDIES DISTRIBUTION NETWORK
Dividends & Working Capital
  • Residual dividend model: pay out what is left after funding all positive-NPV projects
  • Dividend irrelevance (M&M): in perfect markets, policy doesn't matter
  • Signaling: dividend cuts signal distress; initiations signal confidence
  • Cash Conversion Cycle: CCC = DSO + DSI − DPO
  • Cost of trade credit = (Discount% / (1 − Discount%)) × (365 / Days foregone)
  • Repurchases vs. dividends: tax-advantaged when capital gains < income tax
CCC = DSO + DSI − DPO
WEEK 12
MODULE // 11
GLOBAL NET CORRIDORS
International Finance
  • Direct quote: $/FC. Indirect: FC/$. Cross rates via triangular arbitrage
  • CIRP: F = S × (1 + r_d) / (1 + r_f) — forward rate pricing
  • PPP: E[S_t] = S₀ × [(1 + π_d) / (1 + π_f)]^t
  • International NPV: convert foreign FCFs at expected forward rates, discount at domestic WACC
  • Political risk, expropriation, repatriation restrictions
  • Currency hedging with forwards, swaps, money market hedge
CIRP: F = S × (1 + r_d)/(1 + r_f)
WEEK 13
MODULE // 12
DERIVATIVES STREET
Financial Options
  • Call: right to BUY at K; payoff = max(S_T − K, 0)
  • Put: right to SELL at K; payoff = max(K − S_T, 0)
  • Put-call parity: C − P = S₀ − PV(K) [European, no dividends]
  • Binomial: risk-neutral probability p = (e^{rΔt} − d)/(u − d)
  • Real options: expand (call), abandon (put), delay (call)
  • Black-Scholes: C = S×N(d₁) − K×e^{-rT}×N(d₂)
Put-Call Parity: C − P = S₀ − K·e^{−rT}

FORMULA REFERENCE

Every formula you need — know the intuition, not just the equation.

TIME VALUE OF MONEY
Future Value FV = PV × (1 + r)^t
Present Value PV = FV / (1 + r)^t
Annuity PV PV = PMT × [1 − (1+r)^−n] / r
Perpetuity PV PV = C / r
Effective Annual Rate EAR = (1 + APR/m)^m − 1 m = compounding periods per year
RISK & RETURN / CAPM
CAPM / Security Market Line r_i = r_f + β_i × (r_M − r_f)
Beta (regression slope) β = Cov(R_i, R_M) / Var(R_M) Excel: =SLOPE(stock_rets, market_rets)
Portfolio Expected Return E[R_p] = Σ w_i × E[R_i]
Sharpe Ratio SR = (E[R_p] − r_f) / σ_p
BOND & STOCK VALUATION
Bond Price (semi-annual) P = Σ C/(1+r/2)^t + Par/(1+r/2)^2n r = YTM; n = years; C = semi-annual coupon
Gordon Growth Model P₀ = D₁ / (r_e − g) D₁ = D₀(1+g); requires g < r_e
Cost of Equity (Gordon solved) r_e = D₁/P₀ + g
CAPITAL BUDGETING
Net Present Value NPV = Σ FCF_t/(1+r)^t − C₀
IRR (solve for r) 0 = Σ FCF_t/(1+IRR)^t − C₀
Profitability Index PI = PV(FCFs) / |C₀| Accept if PI > 1; rank for capital rationing
EAA (unequal lives) EAA = NPV × r / [1 − (1+r)^−n]
WACC & VALUATION
WACC (3 components) WACC = w_d r_d(1−T) + w_ps r_ps + w_e r_e
Terminal Value (Gordon) TV_n = FCF_{n+1} / (WACC − g)
Enterprise Value EV = Σ PV(FCF_t) + PV(TV)
Equity Value per Share P = (EV − Debt + Cash) / Shares
CAPITAL STRUCTURE (M&M)
M&M I (no taxes) V_L = V_U [irrelevance]
M&M I (with taxes) V_L = V_U + T × D T×D = PV of interest tax shield
M&M II — levered cost of equity r_e = r_0 + (r_0 − r_d)(D/E)(1−T)
Hamada — re-levering beta β_L = β_U [1 + (1−T)(D/E)]
DIVIDENDS & WORKING CAPITAL
Cash Conversion Cycle CCC = DSO + DSI − DPO DSO=AR/Sales×365; DSI=Inv/COGS×365; DPO=AP/COGS×365
Cost of Trade Credit r = Disc%/(1−Disc%) × 365/(Pay−Disc days)
DOL (degree of operating leverage) DOL = (S − VC) / EBIT
DFL (degree of financial leverage) DFL = EBIT / (EBIT − Interest)
INTERNATIONAL FINANCE
Covered Interest Rate Parity F = S × (1 + r_d) / (1 + r_f)
Purchasing Power Parity E[S_t] = S₀ × [(1+π_d)/(1+π_f)]^t
International NPV NPV = Σ [FCF_t(FC) × F_t] / (1+r_d)^t Convert FC cash flows at forward rates, discount at domestic WACC
OPTIONS
Call payoff at expiry C_T = max(S_T − K, 0)
Put payoff at expiry P_T = max(K − S_T, 0)
Put-Call Parity (European) C − P = S₀ − K·e^{−rT}
Binomial risk-neutral probability p = (e^{rΔt} − d) / (u − d)
Binomial option price C = e^{−rΔt} × [p·C_u + (1−p)·C_d]

GROUP PROJECT

Full DCF firm valuation — WACC + 3-scenario FCF model. Special session: Saturday July 25, West LA Campus.

FCF EXTRACTION SEQUENCE

  1. Calculate NOPAT from 10-K income statement NOPAT = EBIT × (1 − T)
  2. Compute net operating working capital NOWC = (Cash + AR + Inv) − (AP + Accruals)
  3. Total net operating capital TNOC = Net PPE + NOWC
  4. Net investment in operating capital ΔTNOC = TNOC_t − TNOC_{t−1}
  5. Free Cash Flow FCF = NOPAT − ΔTNOC

DCF VALUATION SEQUENCE

  1. Forecast FCF for 5 years under 3 growth scenarios FCF_t = FCF_0 × (1 + g)^t
  2. Terminal value in year n TV = FCF_{n+1} / (WACC − g_∞)
  3. Enterprise value EV = Σ[FCF_t/(1+WACC)^t] + TV/(1+WACC)^n
  4. Equity value Equity = EV − Total Debt + Cash
  5. Intrinsic price per share P* = Equity Value / Shares Outstanding

WACC BUILD SEQUENCE

  1. Estimate beta from market data β: 5-yr monthly returns vs. S&P 500 (SLOPE in Excel)
  2. Cost of equity via CAPM r_e = r_f + β × (r_M − r_f)
  3. Cost of debt = YTM on bonds (after-tax) r_d(1−T) = YTM × (1 − tax rate)
  4. Capital structure weights (market values) w_e = E/(D+E+PS); w_d = D/(D+E+PS)
  5. Final WACC WACC = w_d r_d(1−T) + w_ps r_ps + w_e r_e
